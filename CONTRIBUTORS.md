## How to Contribute

Contributions are welcome. Please open an issue or pull request on [GitHub](https://github.com/LuyandaLia/reviewflow).

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Commit your changes and open a pull request
4. All contributors will be listed here
